How it works

Over 275,000 tweets appear every minuite, so it is a little bit difficult to check them all. Luckily we created Twemotion, a free service to work out the current mood of twitter. It analyses the last million tweets for search terms and determines the global mood of twitter.

API

Twemotion uses the Twitter node.js api, a free open source API that provides us with a link to Twitter. This means we have a very fast connection directly to twitter.

OPEN DATA

For the 'Open Data' requirement of the FoC, we have used Twitter. It has miles of data avalible for free, and a wide number of API's avalibile for most tasks.

SEARCH

We have refined our twitter search to use a complex selection of attributes & weightings to determine the mood of twitter. For example: i+hate OR really+angry OR i+am+mad OR really+hate OR so+angry
